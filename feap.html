Installation und Grundlagen

FEAP benötigt das Package xfonts-75dpi, welches bei aktuellen Linux-Distributionen nicht mehr standardmäßig installiert ist. Um das Package zu installieren, muss folgender Befehl im Terminal ausgeführt werden: 

sudo apt-get install xfonts-75dpi

FEAP kann nun aus einem Terminal gestartet werden: 

./feap

Mit dem Parameter -i kann direkt beim Programmaufruf eine Eingabedatei übergeben werden. Dazu muss der Dateiname ohne Leerzeichen angehängt werden. Um beispielsweise eine Datei Ipatch zu öffnen, wird folgender Befehl verwendet: 

./feap -iIpatch

FEAP-Befehle

TANG
Tangentiale Steifigkeitsmatrix berechnen
FORM
Residuum berechnen
SOLVE
LGS lösen, Verschiebungen berechnen
TANG,,1
entspricht der Befehlsfolge TANG FORM SOLVE
DISP,all
Verschiebungen für alle Knoten
DISP,,nfirst,nlast,delta
Verschiebungen für einzelne Knoten
REAC,all
Reaktionskräfte für alle Knoten
REAC,,nfirst,nlast,delta
Reaktionskräfte für einzelne Knoten
STRE,all
Spannungen für alle Elemente
STRE,,nfirst,nlast,delta
Spannungen für einzelne Elemente
PLOT
in den Plot-Modus wechseln
QUIT
FEAP verlassen

FEAP-Befehle im Plot-Modus

WIPE
Plot löschen
POST
Ausgabe in ein Postscript-File feapX.eps (Achtung: dieser Befehl muss vor allen anderen Plot-Befehlen ausgeführt werden)
COLO,i
Farbskalierung des Postscript-Files
MESH
deformiertes/undeformiertes Elementnetz
NODE
Knoten mit Nummern anzeigen
ELEM
Elemente mit Nummern anzeigen
BOUN
Randbedingungen anzeigen
LOAD
Lasten anzeigen
DEFO,,faktor
Wechsel zu deformierter Lage
UNDE
Wechsel zu Ausgangslage
STRE,i
i-te Spannung zeichnen
STRE,1
σxx zeichnen
STRE,2
σyy zeichnen
STRE,3
σzz zeichnen
STRE,4
σxy zeichnen
SPLO
Spannungsverlauf in Schnitt zeichnen
PERS
perspektivische Darstellung
AXIS
Koordinatensystem anzeigen
END
